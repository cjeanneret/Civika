## Principes transverses
- Priorité sécurité dès le PoC.
- Privacy-first: aucun compte, aucun profilage utilisateur.
- Données sources officielles, ouvertes et documentées.
- Mode RAG explicite (`local` ou `llm`) sans fallback implicite.

## Versions linguistiques
Les documents structurants existent en cinq variantes:
- Français: fichier sans suffixe (ex: `development.md`)
- Anglais: suffixe `.en.md`
- Italien: suffixe `.it.md`
- Allemand: suffixe `.de.md`
- Romanche: suffixe `.rm.md`
