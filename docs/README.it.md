## Principi trasversali
- Sicurezza prioritaria fin dal PoC.
- Privacy-first: nessun sistema di account, nessuna profilazione.
- Fonti dati ufficiali, aperte e documentate.
- Modalità RAG esplicita (`local` o `llm`) senza fallback implicito.
