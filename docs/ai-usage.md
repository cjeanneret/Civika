# Usage de l'IA

## Contexte
Cette page documente de manière transparente comment l'IA a été utilisée dans le projet Civika.

## Ce qui a été assisté par IA
- Une part importante du code a été générée avec assistance IA.
- La génération a été faite sous supervision humaine continue.
- L'IA a aussi été utilisée pour assister certaines sessions de debugging.

## Validation et qualité
- Les tests ont été exécutés manuellement.
- La validation fonctionnelle et technique a été réalisée manuellement.
- Les décisions finales d'acceptation restent sous responsabilité humaine.

## Limites et responsabilité
- L'assistance IA peut introduire des erreurs, régressions ou interprétations incomplètes.
- Aucune sortie IA n'est considérée fiable sans revue humaine explicite.
- En cas d'écart, la référence est le comportement vérifié manuellement.

## Privacy et données
- Les sessions d'assistance IA n'ont pas pour objectif de traiter des données personnelles utilisateur.
- Le projet reste aligné avec les principes privacy-first du repository.

## Traçabilité
- Dernière mise à jour: 2026-03-14.
- Périmètre concerné: backend, frontend, documentation et debugging.
