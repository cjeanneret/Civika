## Übergreifende Prinzipien
- Sicherheit hat Priorität ab dem PoC.
- Privacy-first: kein Kontosystem, kein Nutzerprofiling.
- Offizielle, offene und dokumentierte Datenquellen.
- Expliziter RAG-Modus (`local` oder `llm`) ohne impliziten Fallback.
